# LEYE Take Home Assessment - Raymond Duong

Raymond Duong's submission of LEYE's Spring 2026 Coding Challenge

## Prerequisites

Before running this project, ensure you have the following installed on your machine/physical device:

- Node (v18 or later recommended)
- npm, Yarn, or bun
- Expo Go app on a physical device
- For iOS simulation: macOS with Xcode installed
- For Android simulation: Android Studio with an emulator configured

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rduong11/leye-take-home-assessment
cd leye-blog
```

### 2. Install dependencies

Using npm:

```bash
npm install
```

Using yarn:

```bash
yarn install
```

Using bun:

```bash
bun install
```

### 3. Start the development server

```bash
npx expo start
```

This will open the Expo developer tools in your browser and display a QR code in the terminal.

## Running the App

### On a physical device:

Install the Expo Go app from the App Store (iOS) or Google Play Store (Android).
Scan the QR code displayed in the terminal or browser after running `npx expo start`

### On an iOS Simulator (macOS only)

With Xcode installed, press `i` in the terminal after running `npx expo start`, or click "Run on iOS Simulator" in the Expo browser tools.

### On an Android Emulator

With Android Studio and an emulator configured, press `a` in the terminal after running `npx expo start`, or click "Run on Android device/emulator" in the Expo browser tools.
